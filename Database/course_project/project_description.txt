Тема курсового проекта: "Описание модели хранения данных сайта Кинопоиск"

Таблицы Базы данных:

1. Жанры
2. Фильмы - id фильма, id жанра, название, год выхода, сборы(?), описание
3. Киноличности - актеры, режисеры (id актера, ФИО, id профессии(актер, режисер, актер/режисер) биография(?))
4. Профессии в кино - id профессии(актер, режисер, актер/режисер)
5. Фильмографии - связь М*М между таблицами "Фильмы" и "Киноличности" (id фильма, id актера/режисера/продюссера)
6. Пользователи профиль
7. Избранные фильмы (закладки) (id записи, id пользователя, id фильма)
8. Рецензии (id записи, id пользователя, id фильма, текст)
9. Рейтинги - связь М*М между таблицами "Пользователи" и "Фильмы" (id пользователя, id фильма, оценка)
10. Media - инф о медиа файлах (файлы фильма, трейлеры, фото актеров, кадру из фильмов)
