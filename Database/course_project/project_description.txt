Тема курсового проекта: "Описание модели хранения данных сайта Кинопоиск"

Таблицы Базы данных:

1. category Жанры
2. movies Фильмы - id фильма, id жанра, название, год выхода, режисер
3. people Киноличности - актеры, режисеры (id актера, ФИО, биография)
4. professions Профессии в кино - id профессии, название(актер, режисер, сценарист и т.д.)
5. filmography Фильмографии - связь М*М между таблицами "Фильмы" и "Киноличности" (id фильма, id актера/режисера/продюссера)
6. users Пользователи (осн. инфа для авторизации) 
7. profiles Профиль пользователя (доп. инфа)
8. bookmarks Избранные фильмы (закладки) - связь М*М (id пользователя, id фильма)
9. reviews Рецензии (id записи, id пользователя, id фильма, текст)
10. rating Рейтинги - связь М*М между таблицами "Пользователи" и "Фильмы" (id пользователя, id фильма, оценка)
11. мedia Общая инф о медиа файлах
12. media_types Описание типов медиа файлов в таблице media (трейлер, афиша, кадры из фильма, описание фильма-description, фото актера)
13. мedia_movies Медиа файлы связанные с фильмом (трейлер, афиша, кадры из фильма, файл на фильм, описание фильма)
14. мedia_people Медиа файлы связанные с киноличностями (фото, биография)
15. people_professions - связь М*М между таблицами "Киноличности" и "Профессии в кино"
